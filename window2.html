пример адаптивного сайта Cat Energy, сделанного Павлом

CAT ENERGY

сайт
Описание проделанных работ.

Версия для Desktop, tablet и mobile. Состоит из главной страницы, страницы - каталога и страницы с формами для получения данных от посетителя сайта. Верстка - HTML с использованием БЭМ, стилизация - препроцессоры Less (возможно сделать и с Sass) с использованием Grid и Flexbox, подключение JS для вызова меню по клику кнопки в мобильной версии (при отключенном JS меню также доступно, в этом случае оно открыто по умолчанию и находится в потоке документа). Адаптивная верстка с изменением макетов страниц под разные разрешения ширины экрана (320, 768, 1440) и автоматизация сборки. Оптимизация, ренитизация, кадрирование изображений сайта, в т.ч. адаптация основных изображений под экраны retina. Кроссбраузерность: IE10+, Chrome, Firefox, pera, Safari. Реализована работа элементов, с которыми взаимодействуюет пользователь (ссылки, кнопки, чекбосы форм) и их различное визуальное отображение при разных состояниях.
